Common Tasks

Create a countdown

Use New Countdown for a blank countdown, Quick Add for natural-language text, or import an upcoming Calendar event, dated Reminder, Contacts birthday, Contacts anniversary, or calendar invite file.

Add widgets

Open the in-app Widget Gallery, then add an Orbit Down widget from the system picker. Choose Countdown and optional Layout for one countdown, Source/Collection/Rows for an upcoming list, or Collection for a dashboard. Widgets inherit saved appearances and open the matching countdown when tapped.

Use Siri and Shortcuts

Use App Shortcuts to create countdowns from spoken or typed text, open Quick Add, import Calendar and Reminders items, read summaries, export countdown files, and open Widget Gallery.

Protect your data

Countdowns stay local by default. Optional sync stores individual records and photo assets in your private CloudKit database. Legacy iCloud data is read only during the one-release migration window. You can export backups and use Erase Local Data from Settings when you want to remove local app data.

Troubleshooting

Calendar, Reminders, or Contacts import is empty

Check system privacy permissions for Orbit Down, then make sure the source item has a future date. Calendar and Reminders imports only read items when you choose an import flow.

Widgets are not updating

Open Orbit Down once after editing countdowns, then wait for the system widget refresh window. iOS, iPadOS, and macOS control widget update timing to protect battery life.

Notifications do not appear

Confirm notifications are enabled in system settings. Reminder notifications are scheduled locally and can include optional Time Sensitive delivery when you choose that setting.

Contact

Support contact: https://github.com/Z3lnick/orbitdown/issues/new

Include your device type, OS version, a short description of what happened, and whether the issue involves widgets, Calendar, Reminders, notifications, iCloud sync, backups, or Shortcuts.
